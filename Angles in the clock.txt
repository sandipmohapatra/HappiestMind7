Angles in the clock
 ----------------------------------------------------
 Integer H denotes the current hour on the clock.
 Integer M denotes the current minute on the clock.
 
 Calculate the angle between the hour and minute hand of the clock. 
 
 Note
 The calculated angle should not be more than 180°. Calculate the smallest angle between them.
 
 Function Description
 In the provided code snippet, implement the provided smallestAngle(...) method using the variables to print an integer A denoting the smallest angle between the hour and minute hand. You can write your code in the space below the phrase “WRITE YOUR LOGIC HERE”. 
 
 There will be multiple test cases running so the Input and Output should match exactly as provided.
  
 Input Format
 The first line contains the integer H, denoting the current position of the hour hand.
 The next line contains the integer M, denoting the current position of the minute hand.
 
 Sample Input
 00 -- denotes H
 20 -- denotes M
  
 Output Format
 The output returns an integer A denoting the smallest angle between the hour and minute hand.
 
 Sample Output
 120
 
 Explanation
 The hour hand is pointing at 12.
 The minute hand is pointing at 4.
 Each clock hand will cover 360° and each clock has 60 minutes.
 Therefore, 360° total ÷ 60 minutes total = 6° per minute
 Hence 6° * 20 minutes =120°
 The angle formed between them is 120.
-------------------------------------------------------------------------------------------------------------
import java.util.*;
 import java.lang.*;
 import java.io.*; 
 public class Main {
  public static int Angle(int H, int M) {  
  int A=0;  
  //write your Logic here:
//handle 24-hour notation
  H = H % 12;

  // find the position of the hour's hand
  int h = H*30;

  // find the position of the minute's hand
  int m = (M * 6);

  // calculate the angle difference
  int angle =Math.abs(h - m);

  // consider the shorter angle and return it
  if (angle > 180)
  {
      angle = 360 - angle;
  }
    A = angle;
    return A;
  }
  public static void main(String[] args) {
  Scanner sc=new Scanner(System.in);  
  int H=sc.nextInt();
  int M=sc.nextInt();  
  int A=Angle(H,M); 
  System.out.println(A);
  sc.close();
  } 
 }
